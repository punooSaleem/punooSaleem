About Web based Calculator


This project is a Simple Web based calculator developed using different technologies
to perform basic Arithmetic operations like: Addition, Subtraction,
multiplication and division. This report details the design, development,
and testing of the calculator.

introduction

The purpose of this project is to create a functional calculator using web technologies.
This project serves as a demonstration of HTML for structure,
CSS for styling, and JavaScript for functionality. Technologies Used
• HTML
• CSS
• JavaScript
• VS Code
• BootStrap
